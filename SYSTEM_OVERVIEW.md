# ระบบจัดการผลผลิตทางการเกษตร - Agricultural Product Management System

## โครงสร้างระบบ

### หน้าหลักทั้งหมด (5 หน้า)
1. **Dashboard (ภาพรวม)** - `/`
2. **Inventory (จัดการสต็อก)** - `/inventory`
3. **Planting Calendar (ปฏิทินการปลูก)** - `/calendar`
4. **Price Analysis (วิเคราะห์ราคา)** - `/analysis`
5. **Recommendations (คำแนะนำ)** - `/recommendations`

## ฟีเจอร์หลัก

### 1. การจัดการข้อมูลผลผลิต ✅
- เพิ่ม แก้ไข ลบ ข้อมูลผลผลิต (Admin only)
- ข้อมูลประเภทผลผลิต: ข้าว ผักสด ผลไม้ พืชผล ถั่ว-งา เครื่องเทศ สมุนไพร
- จัดเก็บปริมาณผลผลิต พร้อมหน่วยนับที่หลากหลาย (กิโลกรัม ลูก หวี กำ ลัง ตัน ถัง)
- บันทึกวันที่เก็บเกี่ยวและแหล่งที่มา (ผู้จัดจำหน่าย)
- ระบบแจ้งเตือนสินค้าใกล้หมด

### 2. การจัดการฤดูกาลและการปลูกพืช ✅
- บันทึกข้อมูลฤดูกาลเพาะปลูก (ฝน หนาว ร้อน)
- ระบุช่วงเดือนที่เหมาะสมต่อการปลูกพืชแต่ละชนิด
- บันทึกระยะเวลาการปลูกจนถึงการเก็บเกี่ยว
- บันทึกพื้นที่การปลูก (ไร่) และผลผลิตโดยประมาณ (กก.)
- ติดตามสถานะ: วางแผนแล้ว / กำลังปลูก / เก็บเกี่ยวแล้ว

### 3. การจัดการข้อมูลราคา ✅
- จัดเก็บข้อมูลราคาพืชในแต่ละช่วงเวลา (6 เดือน)
- แสดงระดับราคา: ต่ำ / ปานกลาง / สูง
- เปรียบเทียบราคาพืชตามเดือน
- กราฟแนวโน้มราคาและการวิเคราะห์
- คำนวณราคาเฉลี่ย ราคาสูงสุด และราคาต่ำสุด

### 4. การค้นหาและการแนะนำ ✅
- ค้นหาข้อมูลผลผลิตตามชื่อ ประเภท หรือแหล่งที่มา
- กรองข้อมูลตามหมวดหมู่
- แสดงข้อมูลผลผลิตทั้งหมดในระบบ
- แนะนำพืชที่เหมาะสมตามฤดูกาลปัจจุบัน
- แนะนำพืชที่มีราคาสูง (คุ้มค่าปลูก)
- แนะนำช่วงเวลาที่ราคาดีที่สุดสำหรับการขาย

### 5. User Role Management ✅
- **Admin**: สามารถ CRUD ข้อมูลทั้งหมด
- **User**: สามารถดูข้อมูล ค้นหา และรับคำแนะนำ (Read-only)

## ข้อมูลตัวอย่างในระบบ

### ผลผลิต
- ข้าวหอมมะลิ
- มะม่วงน้ำดอกไม้
- ผักกาดหอม
- มะเขือเทศ
- กล้วยหอม
- มันฝรั่ง

### ข้อมูลฤดูกาล
**ฤดูฝน** (มิ.ย. - ต.ค.)
- ข้าวหอมมะลิ, ข้าวเหนียว, ข้าวโพด, ถั่วเหลือง

**ฤดูหนาว** (พ.ย. - ก.พ.)
- ผักกาดหอม, มะเขือเทศ, แตงกวา, บรอกโคลี, กะหล่ำปลี

**ฤดูร้อน** (มี.ค. - พ.ค.)
- มะม่วง, ทุเรียน, ส้มโอ, ลำไย, มะละกอ

## วัตถุประสงค์
1. จัดเก็บและบริหารจัดการข้อมูลผลผลิตทางการเกษตรอย่างเป็นระบบ
2. ช่วยวางแผนการปลูกพืชให้เหมาะสมกับฤดูกาล
3. วิเคราะห์และแสดงช่วงเวลาที่พืชมีราคาดี
4. สนับสนุนการตัดสินใจของเกษตรกรเพื่อเพิ่มผลผลิตและรายได้

## การใช้งาน
1. เข้าใช้งานระบบผ่านเบราว์เซอร์
2. นำทางผ่านเมนูด้านซ้าย (Desktop) หรือปุ่ม Menu (Mobile)
3. ผู้ใช้สามารถจัดการข้อมูลทั้งหมดได้ (เพิ่ม แก้ไข ลบ)
4. ข้อมูลทั้งหมดถูกจัดเก็บแบบ Local Storage
